Glossário HTML

Na linguagem HTML utilizamos as "tags", utilizadas para: destacar; rotular, usadas para que quando o navegador as rotular nos site, apareça de forma ordenada.

Dentre as tags que já estudamos, temos:

  1. < body> < body> que é referente ao corpo do texto
  2. < p> < /p> que é referente ao parágrafo
  3. < br> < br> referente à quebra de texto
  4. Também podemos mudar os tamanhos do texto usando as tags:
    • < h1> < /h1> para o título
    • < h2> < /h2> para o subtitulo
    • e < h3> < /h3>; < h4> < /h4>; < h5> < /h5> e < h6> < /h6> para os demais textos, lembrando que conforme o número usado for maior, o texto vai diminuindo.
  5. < strong> < /strong> referente ao negrito
  6. < hr> para uma linha horizontal
  7. < pre> < /pre> registra qualquer formatação ou caractere inserido nele
  8. < em> < /em> deixa o texto em itálico
  9. < ins> < /ins> deixa o texto sublinhado
  10. < del> < /del> utilizado para tachar
    • E para tachar também podemo usar < s> < del> tachar
  11. < q> < /q> utilizado para fazer pequenas citações
  12. < blockquote> < /blockquote> para fazer longas citações
  13. < sup> < /sup> utilizado para colocar números em cima2
  14. < sub> < /sub> utilizado para adicionar números em baixo2
  15. < ol> < /ol> referente à listas ordenadas (deve ser utilizado antes do < li>)
  16. < ul> < /ul> lista não ordenada (deve vir antes do < li>)
  17. < li> < /li> referente à lista
  18. < a> < /a> utilizado para criar links, e é necessário adicionar o atributo href, que mostra onde o navegador deve direcionar o usuário caso o link seja clicado: < a href="linl que você deseja utilizar"> < /a>

    19. Na linguagem também utilizamos os comandos que são objetos identificáveis dentro de um documento, que podem resultar em uma imagem, uma tabela, ou qualquer outro elemento visível.

    Dentre os comandos que já estudamos temos:

    1. "!" sempre devemos começãr com esse comando para iniciar nosso código, devemos sempre lembrar de mudar o idioma para "pt-br"
    2. "LoremX" é utilizado para testar, demonstrar as tags que utilizamos e ver se ficou como queríamos (Lorem= conjunto aleatório de palavras e X= número de palavras que você quer)
    3. type= “I”: transforma listas em números romanos (esse comando pode variar para dezenas de formas, como exemplo Circle, square etc)
